# Sjekksum

A gem to provide some checksum algorithms like Luhn, Damm and Verhoeff.



## Installation

Add this line to your application's Gemfile:

    gem "sjekksum"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install damm
